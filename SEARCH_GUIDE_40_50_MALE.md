# 40-50대 남성 구매 키워드 검색 가이드

## 생성된 키워드 파일
`keywords_40_50_male.txt` 파일에 40-50대 남성들이 주로 구매하는 상품 키워드가 포함되어 있습니다.

## 검색 실행 방법

### 1. 도매꾹에서만 검색 (기본)
```bash
python main.py --file keywords_40_50_male.txt --max-results 20 --format csv
```

### 2. 모든 사이트에서 검색
```bash
python main.py --file keywords_40_50_male.txt --search-all-sites --max-results 20 --format csv
```

### 3. 가격 필터 적용
```bash
# 최소 가격 10,000원 이상
python main.py --file keywords_40_50_male.txt --min-price 10000 --max-results 20

# 가격 범위 지정 (10,000원 ~ 200,000원)
python main.py --file keywords_40_50_male.txt --min-price 10000 --max-price 200000 --max-results 20
```

### 4. 헤드리스 모드
```bash
python main.py --file keywords_40_50_male.txt --headless --max-results 20 --format csv
```

## 30-40대와의 차이점

40-50대 남성 키워드는 다음과 같은 특징이 있습니다:
- **건강관리**: 혈압계, 혈당측정기 등 건강 모니터링 제품
- **고급 의류**: 정장, 수트 등 비즈니스 의류 중심
- **취미 활동**: 골프, 낚시 등 여가 활동 관련 제품
- **실용성**: 자동차용품, 사무용품 등 실용적인 제품
- **안티에이징**: 스킨케어, 안티에이징 제품

## 검색 후 작업

### 1. 결과 파일 병합
```bash
python main.py --merge-csv
```

### 2. 통합 파일에서 상품 등록
```bash
python main.py --register-from-file merged_results_YYYYMMDD_HHMMSS.csv --max-register-products 100
```
